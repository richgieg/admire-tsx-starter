# admire-starter

Starter project for static website using Admire with TSX add-on

## Clone
```
npx degit richgieg/admire-tsx-starter my-new-project
cd my-new-project
```

## One-time preparation
```
npm install
```

## Compile and watch for changes
```
npm start
```

## Serve www directory
```
npm run serve
```
